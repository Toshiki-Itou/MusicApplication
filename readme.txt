MusicApplication	ver.1.00

【製作者：伊藤　俊輝】


■諸注意
サーバーからデータを取得しているため現行ディレクトリ内の ≪ public_html ≫ をサーバー側にあげて頂きそこにアクセスするか、または
Bracketsのライブプレビューを使用して ≪ public_html ≫ の直下にある ≪ index.html ≫ を起動してください。

また、音声が再生されるので音量に注意して使用してください。

■動機
音楽仲間と話していた時にこういうソフトあったらいいよねという話になりそれがキッカケで製作を始めました。
また、私自身もベースをやっていて自分の欲しい機能なども取り入れて製作しました。

■工夫点
メニュー：見やすくシンプルを意識しました。
メトロノーム：タイマーや拍数調整など使い勝手の良さを意識しました。
ドラム：直感的に使えるようなデザインにすることを意識しました。
ベース：ベースをやったことがない人でも使えるようにすることを意識しました。
　　　　また音は全て自分で録音した物をつかっています。

■実装できなかった点
ドラム、ベース：三連符や五連符などの特殊な連符
	　　フレーズのセーブ機能、ロード機能

実装できなかった点ではないのですが全て別々に製作していた為、ページごとで大きさが変わったりと統一感がなくなってしまった。
	→ｃｓｓをまとめられるところはまとめる。
